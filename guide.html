テンプレート作成の手順：

  • Figma で「Frame」ツール (F キー) を選択してフレームを描く

  • 「Text」ツール (T キー) で小さめの説明文テキストを配置（例: 14px）

よくある質問・Tips

Q. 見出しの数と説明文の数が違ったら？

少ない方の数に合わせてバナーが生成されます。例：見出し20本、説明文15本なら15枚生成。

Q. 20本以上作りたい場合は？

Claude Code に指示する際に「50本作って」と本数を指定すればOKです。

Claude Code
headline-writer と description-writer を使って、
見出し50本、説明文50本を作ってください。
商品: ...

Q. テンプレートのデザインを変えたい

Figma のテンプレートフレーム内のデザイン（色・画像・レイアウト）は自由に変更できます。テキストノードが2つ以上あればプラグインは動きます。

Q. 列数と間隔って何？

生成されたバナーをどう並べるかの設定です。

  • 列数: 横に何枚並べるか（デフォルト: 5列）
  • 間隔: バナー同士のスキマ（デフォルト: 40px）
